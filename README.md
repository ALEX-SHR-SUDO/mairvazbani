# MairVazBani — Car Services Website

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/ALEX-SHR-SUDO/mairvazbani)

A multilingual static website for an automotive services company based in Israel.

## About

MairVazBani offers a full range of professional car customisation and protection services. The site is available in **three languages** — English, Hebrew (RTL), and Russian — with a language selector in the top-right corner.

## Services

- Window tinting
- Vinyl wrap
- Designer / custom wraps
- Anti-gravel / PPF (paint protection film)
- Headlight restoration
- Nano ceramic coating
- Car audio & multimedia systems

## Languages Supported

| Language | Code |
|----------|------|
| English  | EN   |
| עברית (Hebrew) | HE |
| Русский (Russian) | RU |

## Tech Stack

- **HTML5** — semantic markup
- **CSS3** — dark automotive theme (black & gold), responsive layout
- **Vanilla JavaScript** — language switching, UI interactions

## Deploying to Vercel

This is a pure static site — no build step or package.json required.

### Option 1 — Vercel Dashboard (recommended)

1. Go to [vercel.com](https://vercel.com) and sign in.
2. Click **"Add New Project"**.
3. Select the `ALEX-SHR-SUDO/mairvazbani` repository.
4. Click **Deploy** — the site is live immediately. 🎉

Every subsequent push to `main` will trigger an automatic re-deployment.

### Option 2 — Vercel CLI

```bash
npm i -g vercel
vercel
```

Run the second command from the project root and follow the prompts.
